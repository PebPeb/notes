

- ***Label*** - Is the target of a model so for example if the model is trying to predict what the image is of then the label would be what that image is.
  - Example - A picture of a cat the label would be cat 
- ***Feature*** - Is an individual numerical value that make a input into the model
  - Example - A 64x64 pixel RGB image would have 3 x 64 x 64 features.
